### What is PEP 8 ?
  `PEP = Python Enhancement Proposal`

  * PEP 8 is the official Python style guide that helps developers write readable and maintainable code.

### Key PEP 8 Rules

- Use snake_case for variables and functions.
- Use PascalCase for classes.
- Use spaces around operators.
- Use 4-space indentation.
- Keep names meaningful and concise.
- Use blank lines to improve readability.
- Use UPPER_CASE for constants.
- Write clear comments when necessary.
